# Анализ сетевого трафика: инцидент BURNINCANDLE

Данный репозиторий содержит полный разбор упражнения BURNINCANDLE с платформы Malware-Traffic-Analysis.net. Целью анализа было обнаружение и документирование признаков компрометации в предоставленном сетевом дампе (PCAP).

## Краткие выводы

В ходе анализа обнаружена активность ботнета семейства **EnemyBot/Gafgyt**. Основные находки:
-   **Скомпрометированный хост:** `10.0.19.14` (DESKTOP-SQS305D.bumincandle.com)
-   **Канал управления (C2):** Установлен через аномальный TCP-трафик на внешние адреса. Конечная точка C2 атрибутирована через VirusTotal.
-   **Дополнительная активность:** Обнаружен DDoS-трафик через протокол NetBIOS (порт 137/udp).
-   **Индикаторы компрометации (IoC):** Извлечено более 15 уникальных артефактов (IP-адреса, доменные имена, хэши).

## Методология анализа

Анализ проводился по следующему пайплайну:
1.  **Первичный обзор в Wireshark:** Фокус на аномалиях протокола (TCP retransmission, нестандартные флаги) для выделения подозрительных сессий.
2.  **Выделение ключевых хостов в NetworkMiner:** Идентификация хостов по именам (например, `DESKTOP-SQS305D.local`) и извлечение связанных с ними MAC-адресов и открытых портов.
3.  **Глубокий анализ трафика скомпрометированного хоста:** Применение фильтров в Wireshark по конкретному IP для изучения всех его сетевых взаимодействий.
4.  **Выявление C2-канала:** Поиск зашифрованных или нестандартных TCP-потоков, ведущих на внешние IP-адреса.
5.  **Верификация через Threat Intelligence:** Проверка всех подозрительных IP-адресов и доменов через VirusTotal для атрибуции угрозы.
6.  **Расширение границ инцидента:** Поиск других хостов в дампе, взаимодействующих с обнаруженной C2-инфраструктурой.

## Использованные инструменты

-   **Wireshark** — анализ сетевых пакетов и фильтрация трафика.
-   **NetworkMiner** — извлечение артефактов, анализ хостов и сессий.
-   **VirusTotal** — проверка репутации и атрибуция индикаторов угроз.
-   **Текстовый редактор (Obsidian/VS Code)** — ведение записей и составление отчёта.

## Как использовать этот репозиторий

Материалы предназначены для:
-   Изучения практического подхода к анализу сетевых дампов.
-   Ознакомления с реальными индикаторами компрометации ботнета.
-   Использования в качестве примера для начинающих аналитиков SOC и специалистов по Threat Intelligence.
